Code Generation & Optimization Using IBM Granite

Penyelenggara: IBM SkillsBuild

Tahun: 2025

Sertifikat pengakuan atas komitmen profesional dalam menguasai konsep Code Generation dan Optimization menggunakan IBM Granite. Diterbitkan oleh IBM SkillsBuild dan dapat diverifikasi melalui Credly.

Use Generative AI for Software Development

Penyelenggara: Hacktiv8

Tahun: 2025

Sertifikat penyelesaian kursus “Use Generative AI for Software Development” yang berfokus pada penerapan kecerdasan buatan generatif untuk pengembangan perangkat lunak.

IBM Granite Models for Software Development

Penyelenggara: Hacktiv8

Tahun: 2025

Sertifikat penyelesaian kursus tentang pemanfaatan model IBM Granite dalam pengembangan perangkat lunak modern, berfokus pada integrasi AI dengan proses coding.

Mikrotik

Penyelenggara: Mikrotik, FT-UNP

Tanggal: 25 Oktober 2025

Sertifikat MTCNA (MikroTik Certified Network Associate) adalah sertifikasi tingkat dasar dari MikroTik yang menunjukkan pemahaman dan penguasaan keterampilan dasar dalam konfigurasi dan pengelolaan perangkat jaringan MikroTik. Sertifikasi ini berstandar internasional dan diakui sebagai tolak ukur kompetensi untuk posisi awal di bidang jaringan komputer.

Functions for Manipulating Data in PostgreSQL

Penyelenggara: DataCamp

Tanggal: 9 Oktober 2025

Sertifikat yang membahas cara membuat dan mengelola fungsi PostgreSQL untuk memproses serta memanipulasi data secara efisien.

PostgreSQL Summary Stats & Window Functions

Penyelenggara: DataCamp

Tanggal: 4 Oktober 2025

Sertifikat penyelesaian kursus tentang penggunaan Summary Statistics dan Window Functions pada PostgreSQL untuk analisis data tingkat lanjut.
